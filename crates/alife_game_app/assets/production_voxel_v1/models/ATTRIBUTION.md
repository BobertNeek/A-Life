**Quirky Series - FREE Animals** by **Omabuarts Studio**. They remain
developer-only inputs for the generic future-mesh slicing pipeline.

The named-part OBJs in `../creature_parts/generated/` and the `T_*.png` surface
maps in this directory are original deterministic A-Life production assets
generated by `scripts/generate_canonical_creature_parts.py`. They use the
repository's MIT license. They replace malformed source-animal slices with
canonical biped geometry and do not copy source triangles or source textures.

Keep the Omabuarts attribution with the source OBJ inputs and catalog source
records. Keep the production manifest records for both original generated
runtime assets and licensed developer-only sources when redistributing them.
