# A-Life Future Research Compatibility

Status: non-requirements appendix. This document defines compatibility targets only.

## Purpose

A-Life should not trap itself in a small-animal simulation architecture. The near-term project is an ecosystem game, but the data contracts should also support ascended companions, scalable large brains, founder cohorts, school worlds, dream/replay, and future distributed compute backends.

## Compatibility Targets

1. A favorite evolved creature can be exported from an ecosystem and migrated into a larger brain class.
2. Sensory and action ABIs remain stable across migration.
3. The old evolved core is frozen first, then integrated through gated unfreezing.
4. Expanded lobes include language cortex, working memory, episodic expansion, symbolic scratchpad, math/quantity, narrative/history, social reasoning, self-critic, planning/dream, and speech/writing motor regions.
5. A first serious prototype target is `Student131k`.
6. A first serious school candidate target is `Ascended1M`.
7. `Ascended5M` and beyond are future research classes.
8. Teacher LLMs remain external social actors.
9. Internal semantic priors remain bounded private modules.
10. Backend abstraction allows future multi-GPU or cluster backends without changing creature semantics.

## Ascended Pet Mode

Ascended pet mode is not v0/v1. It is a long-term primary dream for the project. A creature selected from many generations can be converted into a dedicated companion candidate. Its genome, instincts, habits, chemistry, and memory are preserved. New neural territory is added around it.

The old core remains active enough to preserve identity. The new cortex initially shadows and predicts the old core. Over training, the new cortex gains influence by proposing actions through normal arbitration. It may override fear or hunger only when other motivations outweigh them.

## School Candidate Mode

School mode requires a staged curriculum. Do not teach advanced abstractions until the creature has enough context. Early learning is object labels, actions, safety, social signals, and simple cause-effect. Later learning includes reading, writing, math, history, science, tool use, self-reporting, and teaching others.

Math and logic use exact deterministic verifiers. History begins with simulated world history and later may include real human history.

## Non-Requirements

This appendix does not require AGI, human-level cognition, frontier scale, TPU pods, trillion-neuron simulation, or human-plus claims. Formal language should use “grounded developmental generalist agent research direction.”

# Frontier Speculative Notes

This project may eventually be discussed in terms of much larger systems, but this file is explicit: frontier-scale ideas are not implementation requirements. If a future lab wants multi-GPU, distributed, TPU, or human-scale experiments, they must implement `NeuralComputeBackend` without changing creature semantics. The same sensory/action/experience/save contracts should hold. The same distinction between internal semantic prior and external teacher must hold. The same requirement for ablations must hold.

# Companion Ethology Notes

An ascended companion should retain a coherent personality. Preserving identity means preserving the evolved core, chemistry, and action tendencies before adding expansion lobes. The expanded cortex can weigh, reinterpret, and override instincts through arbitration, but not erase them. Fear, hunger, attachment, and curiosity remain felt pressures.

# Scaling Path

Recommended research scale path: 32k toy language prototype, 131k serious prototype, 1M serious school candidate, 5M advanced companion candidate. Larger classes require separate profiling and are not consumer ecosystem defaults.
